# v0.1.2 Release Draft

## Summary

v0.1.2 introduces containerized cartridge launching so a cartridge can ship its own runtime instead of depending on host-installed binaries.

## Highlights

- `tendos run` now supports `launcher.type: docker`
- optional Docker build step from cartridge-owned Dockerfile
- container run passthrough for env vars, volumes, and ports
- dogfood cartridge (`examples/hermes-local-agent`) now runs in a self-contained Docker runtime with in-container Ollama and hermes-compatible entrypoint
- lighter default model for dogfood (`qwen2.5:0.5b`)
