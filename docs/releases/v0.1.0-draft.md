# Tendos v0.1.0 (Draft Release Notes)

## Highlights

Tendos v0.1.0 delivers the foundational cartridge packaging and trust layer for a local-first AI cartridge ecosystem.

- Cartridge specification and manifest validation
- Cartridge pack/unpack workflows
- Ed25519 signing and verification
- Encryption and token primitives for secure distribution/usage
- CLI foundation for init/validate/pack/sign/verify

## Vision Update

This release clarifies Tendos direction as:

"Docker for AI agents and harnesses" — portable, signed cartridges that run local-first and can evolve toward hosted cloud deployment.

## Quality and Security Gates

v0.1.0 is verified through:

- Ruff lint + format checks
- Mypy strict type checks
- Bandit static security scan
- Pytest with coverage gate
- Pre-commit full run
- Build verification (sdist + wheel)
